Component Key
LEGO SPIKE Prime HubMain brain. ARM9 processor, runs MicroPython. 6 sensor/motor ports, Bluetooth, USB-C charging.
LMS-ESP32 v2.0Co-processor bridge board. Plugs into SPIKE via UART. Provides WiFi, extra GPIO, I2C, and ESP32 processing power.
HuskyLens v2AI vision camera. Face recognition, object tracking, color/line detection. Connected to LMS-ESP32 via I2C or UART.
Ultrasonic × 2Left + Right sensors. SPIKE Prime distance sensors. Port A (left) and Port B (right) for dual-side obstacle detection.
Pen Arm + ServoLego Technic beam arm. Servo lifts/lowers pen. Controlled via LMS-ESP32 PWM pin.
DC Drive MotorsLeft + right independent drive. H-bridge motor driver on ESP32 side for speed/direction control.
